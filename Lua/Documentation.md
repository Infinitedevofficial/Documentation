### Menu.CreateMenu


#### Parameters:

| Name | Type | Description |
| :--- | :--- | :--- |
| Position | Vector2D | start position |
| Size | Vector2D | size of the menu |
| Name | string | name of the menu |
| Closable | bool | if true the menu can be closed |



